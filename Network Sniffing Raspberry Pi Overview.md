## Overview

This repository contains materials from ACO499 — an individualized/independent instruction course at ASU focused on network sniffing and intrusion detection using a Raspberry Pi. The course explored deploying IDS (Intrusion Detection System) tools like Suricata on low-cost hardware for home and small-network security monitoring.

## Project Focus

### Raspberry Pi as a Network IDS Device
The core project involved setting up a Raspberry Pi as a passive network sniffer and IDS node. Key areas covered:

- **Suricata IDS** — An open-source, high-performance IDS/IPS engine deployed on the Raspberry Pi
- **Network sniffing** — Capturing and analyzing network traffic on a home/local network
- **Smart home surveillance** — Applying the IDS concept to monitor IoT and smart home device traffic
- **Threat detection** — Using Suricata rules to detect anomalous or malicious network behavior
